# ND VFX Tools
**版本号：v0.22.3**

## 📦 简介
Unity 特效制作工具集，提供高效的特效预览、组合、筛选和智能需求分析功能。

## ✨ 核心功能

### 1. VFX Composer（特效组合器）
- 🎨 实时预览多个粒子特效
- 🔄 支持播放/暂停/重播控制
- 📐 3D 场景视角实时查看
- 💾 一键生成组合预制体
- 🎲 随机生成特效组合

### 2. VFX Filter（特效筛选器）
- 🔍 快速查找定位完整特效预制体
- 👁️ 按需预览特效（按钮触发）
- 🎞️ 可视范围内启用动态缩略图，不可见项自动回退为非动态缓存策略
- 📥 一键添加到 Hierarchy
- 🏷️ 标签筛选和搜索功能
- 🤖 扫描后基于标签池、路径命名和文件夹语义自动匹配并打标签
- ⭐ **精选特效库**：一键查看开发者预设的优质特效，快速引用

### 3. Favorite Library（精选特效库）🔄 性能优化
- ⭐ 开发者精选的优质特效预制体集合
- 📁 独立 JSON 文件存储，随工具分发
- 🔒 只读模式：使用者只能查看和引用，无法修改
- 👨‍💻 开发者工具：通过「精选库管理（开发者）」窗口管理精选库
- 🚀 快速定位优质特效，节省筛选时间
- 🎞️ 可视范围内启用动态缩略图，不可见项自动回退为非动态缓存策略（与筛选器一致）
- 📄 分页模式：每页 200 项，配合虚拟滚动优化大量特效时的渲染性能
- ➕ 筛选器扫描结果支持一键添加到精选库

### 4. Library Browser（预制体浏览器）
- 📂 可视化浏览项目中的预制体
- 🔍 支持搜索和筛选
- 👁️ 实时预览预制体内容
- ➕ 快速添加到 Composer
- 🏷️ 自动标签识别和筛选

### 4. Tag Pool（标签池）🔄 重构
- 📁 **独立标签池文件**：标签池存储在独立的 JSON 文件中，便于管理和分发
- 🔒 **只读模式**：使用者只能查看和使用标签，无法修改标签池
- 👨‍💻 **开发者工具**：通过「标签池管理（开发者）」窗口管理标签池
- 📥 **导入导出**：支持从外部 JSON 导入标签，保存时自动覆盖标签池文件
- 🎨 标签颜色自定义，新增时自动生成随机颜色

### 6. Requirement Analyzer（需求分析器）🔄 独立窗口
- 🖼️ 支持拖放或选择特效需求截图
- 🤖 AI 全面提取需求关键词（角色/技能/特效类型/视觉风格等）
- 🏷️ 标签池反向筛选匹配，精准锁定相关标签
- ⚡ 分析结果同时应用到筛选器和组合器
- 🧹 支持清空功能，方便重新分析
- 📍 从筛选器/组合器点击按钮打开独立窗口
- 🌋 支持火山引擎豆包视觉 API（按 Gemini 类似方式填写 API Key，默认复用项目内置 Endpoint ID）

## 🚀 安装方法

### 通过 Git URL 安装（推荐）

1. 打开 Unity 编辑器
2. 打开 `Window > Package Manager`
3. 点击左上角 `+` 按钮
4. 选择 `Add package from git URL...`
5. 输入：`https://github.com/ZY-ND/ND-VFX-Tools.git`
6. 点击 `Add`

### 通过本地路径安装

1. 将整个 `VFX Tools` 文件夹复制到目标项目的 `Packages/` 目录下
2. 重命名为 `com.nd.vfxtools`
3. Unity 会自动识别并加载

### 通过 manifest.json 安装

在项目的 `Packages/manifest.json` 中添加：

```json
{
  "dependencies": {
    "com.nd.vfxtools": "https://github.com/ZY-ND/ND-VFX-Tools.git",
    ...其他依赖
  }
}
```

或本地路径：
```json
{
  "dependencies": {
    "com.nd.vfxtools": "file:../本地路径/com.nd.vfxtools",
    ...其他依赖
  }
}
```

## 📖 使用说明

### 打开工具窗口
- 特效组合器：`Tools > VFX > VFX Composer`
- 特效筛选器：`Tools > VFX > VFX Filter`
- 需求分析器：从筛选器或组合器点击「需求分析器」按钮打开
- 标签池管理（开发者）：`Tools > VFX > 标签池管理（开发者）`
- 精选库管理（开发者）：`Tools > VFX > 精选库管理（开发者）`

### 标签池使用说明
- **标签池文件位置**：`Assets/VFX Tools/Editor/VFXTagPool.json`
- **使用者**：直接使用筛选功能，标签池为只读
- **开发者**：通过「标签池管理（开发者）」窗口管理标签
  - 新增/编辑/删除分类和标签
  - 从外部 JSON 导入标签数据
  - 点击「保存」覆盖标签池文件
- **分发**：将 `VFXTagPool.json` 随工具一起分发，其他用户自动获得预设标签

### 精选特效库使用说明
- **精选库文件位置**：`Assets/VFX Tools/Editor/VFXFavoriteLibrary.json`
- **使用者**：在 VFX Filter 中点击「精选特效库」按钮，快速查看开发者预设的优质特效
  - 精选库模式与扫描库模式完全隔离：精选库模式不显示扫描路径与扫描按钮
  - 精选库模式支持标签筛选，便于快速定位
  - 可视范围内启用动态缩略图，不可见项自动回退静态缓存，大量特效时保持流畅
  - 可预览、添加到场景、添加到组合器
  - 支持点击「标签」按钮查看该精选项已绑定标签
  - 无法修改精选库内容
- **从筛选器添加到精选库**：
  - 在扫描库模式中，每条特效右侧有「收藏」按钮，点击即可添加到精选库
  - 已收藏的特效显示「已收藏」状态，防止重复添加
  - 筛选结果上方有「批量收藏」按钮，可将当前筛选结果全部添加到精选库
- **开发者**：通过「精选库管理（开发者）」窗口管理精选库
  - 支持从筛选器扫描结果一键批量添加到精选库
  - 删除不需要的特效
  - 清理失效项（预制体已删除或移动）
- **标签一致性**：当标签池中的标签重命名或删除时，精选库会自动同步更新
- **分发**：将 `VFXFavoriteLibrary.json` 随工具一起分发，其他用户自动获得精选库

### VFX Composer 使用流程
1. 点击 `Open Library Browser` 打开预制体浏览器
2. 在浏览器中选择特效预制体，点击 `Add to Composer`
3. 在 Composer 列表中查看和调整特效
4. 使用播放控制按钮实时预览效果
5. 点击 `Create Combined Prefab` 生成组合预制体

### VFX Filter 使用流程
1. 设置扫描路径，点击 `Scan` 扫描完整特效预制体，并仅使用标签池内已有标签自动打标
2. 自动打标会结合预制体名称、所在文件夹命名和常见语义别名（如技能 / UI / 场景 / 常态 / 通用）进行匹配
3. 列表仅对可视范围内条目启用动态缩略图，以兼顾预览效果与滚动性能
4. 使用搜索框或标签筛选器快速定位特效
5. 点击 `预览` 按钮在 SceneView 中查看效果
6. 点击 `添加到场景` 将特效实例化到 Hierarchy
7. 点击 `标签` 按钮编辑特效标签

### Requirement Analyzer 使用流程
1. 从筛选器或组合器点击「需求分析器」按钮打开独立窗口
2. 点击右上角 ⚙ 按钮打开 API 设置
3. 选择 AI 服务（OpenAI / Claude / Gemini / 火山引擎）并配置鉴权信息
4. Gemini 默认只需 API Key；火山引擎支持默认 Endpoint 模式和自定义 Endpoint ID 模式
5. 拖放或点击上传特效需求截图
6. 点击「分析需求」按钮
7. 查看分析结果和匹配的标签
8. 点击「应用到筛选器和组合器」同时筛选两个窗口

### 快捷操作
- **播放/暂停**：控制所有特效的播放状态
- **重播**：重新播放所有特效
- **删除**：从列表中移除选中的特效
- **清空列表**：移除所有特效

## ⚙️ 系统要求
- Unity 2021.3 或更高版本
- 支持粒子系统（Particle System）
- 网络连接（用于 AI 分析功能）

## 🔧 技术特性
- ✅ 基于 EditorPrefs 的配置持久化
- ✅ Editor 模式下的粒子系统实时模拟
- ✅ SceneView 自动重绘机制
- ✅ 无需进入 Play 模式即可预览
- ✅ 支持多种 AI Vision API（OpenAI / Claude / Gemini / 火山引擎）
- ✅ 标签池反向筛选匹配算法（完全匹配 → 包含匹配 → 相似度匹配）
- ✅ 独立标签池管理，支持预设分发
